## Superscript

Superscript is a number or letter that written above the normal text.

```
10<sup>th</sup>

```
**💻Example :**
```html
<body>
    10th<br/>
    10<sup>th</sup><br/>
</body>
```
**⚙️ Output :**

![Output](output6.png)

## Subscript

Subscript is a number or letter that written below the normal text.

```

```

It provide extra information about tag
  
```

</body>
```
