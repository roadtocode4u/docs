## Iframe 

**💻Example :**
```html
<!DOCTYPE html>
<html >
<head>
    <title> iframe </title>
</head>
<body>
    <h1> Iframe</h1>
   <iframe src="https://www.roadtocode.org/" height="200px" width="200px">

   </iframe>
</body>
</html>
```
